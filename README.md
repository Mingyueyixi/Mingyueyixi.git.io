项目配置了保存自动编译和浏览器自动刷新。`nodemon` 和 `browser-sync` 的使用：

1、安装开发依赖

`npm install --save-dev browser-sync`
`npm install --save-dev gulp`
`npm install --save-dev gulp-nodemon`

2、配置gulpfile.js文件

3、配置nodemon.json文件

4、package.json的script属性增加键值，以方便运行`npm run xxx`
